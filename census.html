U.S. Census Bureau design guidelines

The U.S.Census Bureau wanted to build a website that housed a set of data visualization standards that folks using Census data could reference. As lead visual designer on this engagement, my responsibilities ranged from research and documentation of data visualization best practices, to designing the U.S. Census-branded documentation website.
